🚀 啟動流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 執行 ./start.sh 或 dub run
  2. 系統載入環境變數 (.env)
  3. 初始化 Vibe.d HTTP 伺服器 (port 8080)
  4. 註冊路由 (OAuth, API, 靜態檔案)
  5. 啟動監聽，等待連線

💡 開發提示
  • 修改 source/app.d 後需重新編譯 (dub build)
  • 修改模板/CSS/JS 後重新載入頁面即可
  • 環境變數修改後需重啟伺服器
  • 建議使用 Git 進行版本控制
  • .env 和 service-account-key.json 絕不加入版控
