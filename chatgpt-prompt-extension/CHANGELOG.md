# 更新日誌 | Changelog

## v2.0.0 (2024-12-10)

### 🎉 重大更新

**新增功能**：

- ✨ **標題功能**：每個提示詞現在都有獨立的標題，一目了然
- 🏷️ **分類管理**：可以為提示詞設定分類（例如：工作、學習、程式設計等）
- 📂 **分類分組顯示**：提示詞按分類分組，更有組織性
- 🔍 **搜尋功能**：可以快速搜尋標題、內容或分類
- 🎯 **分類篩選**：可以只顯示特定分類的提示詞
- 📁 **分類折疊/展開**：點擊分類標題可以折疊或展開該分類
- 🎨 **全新 UI 設計**：更現代化的介面，更好的視覺層次

**改進**：

- 📦 **資料結構升級**：從簡單字串陣列升級為物件陣列
- 🔄 **自動遷移**：舊版本資料會自動升級到新格式
- 💾 **增強匯出格式**：匯出檔案包含版本資訊和時間戳記
- 📥 **智慧匯入**：支援匯入新舊兩種格式的資料
- 🎯 **更好的提示詞管理**：新增時自動聚焦，方便連續新增
- ⚡ **效能優化**：分類選項動態更新，渲染更流暢

**技術細節**：

- 新增 `id`、`title`、`category`、`createdAt` 欄位
- 新增資料版本控制（v2.0）
- 新增分類自動完成（datalist）
- 新增即時搜尋過濾
- 新增分類群組展開/折疊狀態記憶

**向下相容**：

- ✅ 完全向下相容 v1.0.0 資料格式
- ✅ 自動偵測並遷移舊資料
- ✅ 支援匯入舊版本 JSON 檔案

## 升級指南

### 從 v1.0.0 升級到 v2.0.0

**方式 1：自動升級（推薦）**

1. 直接載入新版本 Extension
2. 開啟 Popup，系統會自動偵測舊資料
3. 自動遷移完成，舊資料保留為「未分類」分類
4. 開始使用新功能！

**方式 2：手動備份升級**

1. 在舊版本匯出資料
2. 載入新版本 Extension
3. 匯入之前匯出的 JSON 檔案
4. 系統會自動轉換格式

**資料變更**：

舊格式：
```json
[
  "提示詞內容 1",
  "提示詞內容 2"
]
```

新格式：
```json
[
  {
    "id": "unique-id",
    "title": "提示詞標題",
    "content": "提示詞內容",
    "category": "分類名稱",
    "createdAt": 1234567890
  }
]
```
