# 隱私權政策 | Privacy Policy

## 繁體中文

### ChatGPT Prompt 快捷庫 - 隱私權政策

**最後更新日期：2024-12-09**

#### 1. 資料收集

本 Chrome Extension **不會收集、傳輸或儲存任何個人資料到外部伺服器**。

#### 2. 資料儲存

- 所有提示詞資料皆儲存於使用者本機的瀏覽器 `localStorage` 中
- 資料不會上傳至雲端或任何遠端伺服器
- 資料僅存在於使用者的裝置上

#### 3. 權限說明

本 Extension 僅請求以下權限：

| 權限 | 用途 |
|------|------|
| `storage` | 用於在本地儲存提示詞資料 |
| `activeTab` | 用於取得當前分頁資訊，確認是否在 ChatGPT 網站 |
| `scripting` | 用於在 ChatGPT 頁面注入內容腳本，實現插入功能 |
| `host_permissions` | 僅限 `chat.openai.com` 和 `chatgpt.com` 網域 |

#### 4. 資料使用

- 本 Extension 不會追蹤使用者行為
- 不會收集任何分析數據
- 不會讀取使用者的 ChatGPT 對話內容
- 僅在使用者主動點擊時插入提示詞

#### 5. 第三方服務

本 Extension **不使用任何第三方服務**，包括：
- 不使用 Google Analytics
- 不使用廣告服務
- 不連接任何外部 API

#### 6. Cookies

本 Extension **不使用 Cookies**。

#### 7. 兒童隱私

本 Extension 不特別針對 13 歲以下兒童，也不會故意收集兒童的個人資料。

#### 8. 資料安全

- 所有資料儲存在使用者本地瀏覽器
- 使用者可隨時清除瀏覽器資料來刪除所有提示詞
- 建議使用者定期使用「匯出」功能備份資料

#### 9. 隱私權政策變更

如有任何隱私權政策變更，我們會在此頁面更新，並在 Extension 更新說明中通知使用者。

#### 10. 聯絡我們

如對本隱私權政策有任何疑問，請透過以下方式聯絡：
- GitHub Issues
- Email: [您的 email]

## English

### ChatGPT Prompt Manager - Privacy Policy

**Last Updated: December 9, 2024**

#### 1. Data Collection

This Chrome Extension **does not collect, transmit, or store any personal data to external servers**.

#### 2. Data Storage

- All prompt data is stored in the user's local browser `localStorage`
- Data is not uploaded to the cloud or any remote servers
- Data only exists on the user's device

#### 3. Permission Justification

This Extension only requests the following permissions:

| Permission | Purpose |
|------------|---------|
| `storage` | For storing prompt data locally |
| `activeTab` | To get current tab information and verify if on ChatGPT website |
| `scripting` | To inject content scripts into ChatGPT pages for insertion functionality |
| `host_permissions` | Limited to `chat.openai.com` and `chatgpt.com` domains only |

#### 4. Data Usage

- This Extension does not track user behavior
- Does not collect any analytics data
- Does not read user's ChatGPT conversations
- Only inserts prompts when user actively clicks

#### 5. Third-Party Services

This Extension **does not use any third-party services**, including:
- No Google Analytics
- No advertising services
- No external API connections

#### 6. Cookies

This Extension **does not use Cookies**.

#### 7. Children's Privacy

This Extension is not specifically targeted at children under 13 and does not knowingly collect personal information from children.

#### 8. Data Security

- All data is stored in the user's local browser
- Users can delete all prompts by clearing browser data at any time
- Users are recommended to regularly backup data using the "Export" function

#### 9. Privacy Policy Changes

If there are any changes to this privacy policy, we will update this page and notify users in the Extension update notes.

If you have any questions about this privacy policy, please contact us via:
